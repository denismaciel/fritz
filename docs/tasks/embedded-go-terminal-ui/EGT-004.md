# EGT-004: wire embedded interactive mode to internal agent runtime

**Type:** Task

## Goal

Replace the current plain scanner-based local `chat` loop with the embedded Bubble Tea frontend over `internal/agent`.

## Places To Change

- `internal/app/`
- `internal/command/`
- `internal/terminalui/`
- `cmd/fritz/`

## Scope

- local `fritz chat` uses embedded Bubble Tea UI by default
- UI submits prompts to `internal/agent`
- UI receives and applies streamed events through the terminal view-model
- support cancel, reset, quit
- keep remote `--server` mode working

## Non-goals

- browser work
- HTTP protocol changes
- complex selector dialogs

## Design Notes

- keep local embedded path separate from remote AG-UI client path
- first cut can keep one-shot `run` on plain stdout
- UI should own terminal rendering only; agent runtime remains headless

## Acceptance Criteria

- `fritz chat` launches the new embedded terminal UI
- `--server ... chat` still works as remote client mode
- cancel/reset/quit work in the local UI

## Dependencies

- `EGT-003`

## Validation

- `go test ./...`
- `go run ./cmd/fritz chat`
