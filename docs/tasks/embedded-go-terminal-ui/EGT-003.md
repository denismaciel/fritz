# EGT-003: add Bubble Tea shell and transcript/input layout

**Type:** Task

## Goal

Create the Bubble Tea terminal shell with transcript viewport, input area, and basic layout/styling.

## Places To Change

- `go.mod`
- `internal/terminalui/`
- `cmd/fritz/`

## Scope

- add `bubbletea`, `bubbles`, `lipgloss`
- build base Tea model with:
  - transcript viewport
  - multiline input
  - status/footer area if needed
- support resize handling
- support render of static transcript items from the view-model

## Non-goals

- wiring to the live agent runtime
- session/model pickers
- advanced overlays/forms

## Design Notes

- keep styles minimal and readable
- use `textarea` if multiline editing is needed from day one, else `textinput` plus later upgrade
- prefer small component composition over custom rendering tricks

## Acceptance Criteria

- a Bubble Tea model renders transcript + input cleanly in tests/manual smoke run
- dependencies are added cleanly to the module
- no runtime logic moves into UI package

## Dependencies

- `EGT-002`

## Validation

- `go test ./...`
- `go run ./cmd/fritz help`
