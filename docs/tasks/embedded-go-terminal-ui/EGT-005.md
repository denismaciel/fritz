# EGT-005: render compact tool outputs and reasoning blocks

**Type:** Task

## Goal

Make the embedded UI useful for real agent work by rendering tool executions compactly and reasoning blocks separately.

## Places To Change

- `internal/terminalui/`
- `internal/agent/`
- `internal/tool/` if preview helpers belong there

## Scope

- render tool rows with:
  - tool name
  - compact args summary
  - output preview when present
  - clear error indication
- render reasoning blocks distinctly from assistant text
- add collapse/expand or show/hide behavior as needed for large outputs/reasoning
- handle long tool output without flooding the viewport

## Non-goals

- full tree browser
- rich diff viewers
- advanced keyboard customization

## Design Notes

- default should favor compactness
- large tool output should never dominate the screen by default
- reasoning visibility can be toggleable if simple enough

## Acceptance Criteria

- tool activity is visible and readable during a real run
- large tool output is reduced by default
- reasoning appears as separate UI content

## Dependencies

- `EGT-004`

## Validation

- `go test ./...`
- `rg -n "reasoning|tool" internal/terminalui`
