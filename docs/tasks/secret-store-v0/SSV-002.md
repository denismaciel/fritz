# SSV-002

## Goal

Implement typed store API.

## Scope

- `Set(name, value)`
- `Get(name)`
- `List()`
- `Delete(name)`
- atomic writes
- stable ordering

## Rules

- values stored plaintext in v0 file only
- `List()` returns names/metadata, not values
- names are normalized and validated

## Done when

- store tests cover roundtrip, overwrite, delete, list redaction
