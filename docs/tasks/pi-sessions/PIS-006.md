# PIS-006 manual compaction pipeline

## Goal

Implement explicit `/compact`-style behavior: prepare, summarize, append compaction entry, rebuild context.

## Scope

- compaction prep
- cut-point selection
- summary entry append
- state rebuild after compaction
- keep extension hook seam optional, not required

## Non-goals

- auto-compaction
- overflow retry
