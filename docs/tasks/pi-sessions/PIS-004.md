# PIS-004 session discovery, continue, open, fork

## Goal

Implement the file-level session ops Pi relies on for `--continue`, `--session`, and file-based fork.

## Scope

- open session by path
- continue most recent session
- list sessions for cwd
- list all sessions
- fork from another session file into current cwd

## Non-goals

- live runtime replacement
- tree navigation inside one session
